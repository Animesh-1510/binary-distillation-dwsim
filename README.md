# Binary Fractional Distillation using DWSIM
Simulation of binary fractional distillation column using DWSIM

## Project Overview
This project presents the simulation of a binary fractional distillation column for the separation of a Benzene–Toluene mixture using DWSIM. The simulation was performed to study multistage separation, reflux operation, and product purity in a continuous distillation process.

## Objective
To simulate and analyze the separation of a Benzene–Toluene mixture using a fractional distillation column and evaluate the effect of column parameters on product composition.

## Software Used
- DWSIM
- Raoult’s Law Thermodynamic Model

## Feed Specifications

| Parameter | Value |
|---|---|
| Binary Mixture | Benzene–Toluene |
| Feed Composition | Benzene = 0.6, Toluene = 0.4 |
| Feed Flow Rate | 100 kmol/hr |
| Operating Pressure | 1 atm |
| Number of Stages | 21 |
| Feed Stage | 21 |
| Reflux Ratio | 2.31 |

## Simulation Results

| Product Stream | Benzene Composition |
|---|---|
| Distillate Product | 0.9900001 |
| Bottom Product | 0.0099999 |

The simulation successfully achieved high-purity separation of the binary mixture through multistage fractional distillation.

## Files Included
- DWSIM simulation file (`.dwxmz`)
- Process flowsheet screenshots

- Temperature profile plots

## Process Description
The feed mixture enters the distillation column where vapor-liquid equilibrium and repeated condensation-vaporization stages enable separation based on relative volatility. Benzene, being more volatile, is obtained predominantly in the distillate stream while Toluene is collected in the bottom stream.

- Application of vapor-liquid equilibrium concepts
